# API de Usuários com Node.js, Express e Prisma

Este projeto é uma API REST simples para gerenciar usuários, construída com Node.js, Express e Prisma ORM conectado a um banco de dados.

## Funcionalidades

- Listar todos os usuários (`GET /usuarios`)

- Criar um novo usuário (`POST /usuarios`)

- Atualizar um usuário existente (`PUT /usuarios/:id`)

- Deletar um usuário (`DELETE /usuarios/:id`)

## Pré-requisitos

- Node.js (v12 ou superior)

- npm ou yarn

- Banco de dados configurado e Prisma Client gerado (ex: PostgreSQL, MySQL, SQLite)
